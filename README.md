# 大乐透号码预测系统

基于历史数据的智能彩票号码预测网站，使用Python Flask开发。

## 🎯 功能特点

- **自动数据采集**：从官方网站自动获取大乐透历史开奖数据
- **智能预测算法**：基于频率统计的热门和冷门号码推荐
- **实时数据更新**：支持自动检查并更新最新期次数据
- **响应式设计**：适配桌面和移动设备的现代化界面
- **历史数据查询**：查看详细的开奖记录和统计信息

## 📊 技术架构

### 后端技术
- **Flask** - Python Web框架
- **Requests** - HTTP请求库
- **BeautifulSoup4** - HTML解析
- **Pandas** - 数据处理

### 前端技术
- **HTML5/CSS3** - 页面结构和样式

## 🔧 核心功能

### 数据采集模块
- 自动爬取大乐透历史开奖数据
- 数据清洗和格式化处理
- 智能去重和增量更新

### 预测算法模块
- **热门号码**：基于历史出现频率最高的号码
- **冷门号码**：基于历史出现频率最低的号码
- 实时数据分析和统计

### 用户界面模块
- 简约大气的现代化设计
- 实时数据状态显示
- 一键生成推荐号码
- 历史数据表格展示
