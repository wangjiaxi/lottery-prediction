# 大乐透号码预测微信小程序

基于原Flask项目转换的微信小程序版本，提供大乐透号码预测和历史数据查询功能。

## 🎯 功能特点

- **数据状态监控**：实时显示当前数据量
- **智能号码推荐**：基于历史数据的冷热号码分析
- **历史数据查询**：表格形式展示开奖记录
- **响应式设计**：适配不同尺寸的移动设备
- **离线数据缓存**：支持本地数据存储

## 🚀 快速开始

### 环境要求

- 微信开发者工具
- 微信小程序账号（用于发布）
- 后端API服务（原Flask项目）

### 开发步骤

1. **下载微信开发者工具**
   - 访问微信官方开发者工具下载页面
   - 安装并登录开发者工具

2. **导入项目**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 选择 `miniprogram` 目录
   - 填写AppID（测试可使用测试号）

3. **配置API地址**
   - 修改 `app.js` 中的 `apiBase` 配置
   - 修改 `utils/api.js` 中的 `API_BASE` 配置
   - 确保后端API服务正常运行

4. **预览和调试**
   - 在开发者工具中预览效果
   - 使用真机调试功能测试

### 发布流程

1. **代码审核**
   - 确保代码符合微信小程序规范
   - 测试所有功能正常

2. **提交审核**
   - 在开发者工具中点击"上传"
   - 登录小程序管理后台提交审核

3. **发布上线**
   - 审核通过后发布版本
   - 用户可搜索使用小程序
