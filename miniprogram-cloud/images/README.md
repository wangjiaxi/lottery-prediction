# 图标文件说明

请在此目录下放置以下图标文件：

## 必需图标文件

1. **home.png** - 首页图标（未选中状态）
2. **home-active.png** - 首页图标（选中状态）
3. **history.png** - 历史数据图标（未选中状态）
4. **history-active.png** - 历史数据图标（选中状态）

## 图标规格要求

- 尺寸：建议 40x40 像素
- 格式：PNG 格式，支持透明背景
- 颜色：未选中状态建议使用灰色，选中状态建议使用主题色

## 临时解决方案

如果没有图标文件，可以：
1. 修改 `app.json` 移除图标配置，使用纯文本
2. 使用微信小程序自带的图标库
3. 创建简单的占位图标文件

## 占位图标创建

可以使用在线工具生成简单的占位图标，或者使用以下颜色方案：

- 首页图标：未选中 #666666，选中 #ff6b6b
- 历史图标：未选中 #666666，选中 #4ecdc4
